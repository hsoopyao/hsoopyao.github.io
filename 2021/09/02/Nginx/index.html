Location

locationжҢҮд»Өзҡ„еҠҹиғҪжҳҜз”ЁжқҘеҢ№й…ҚдёҚеҗҢзҡ„urlиҜ·жұӮпјҢиҝӣиҖҢеҜ№иҜ·жұӮеҒҡдёҚеҗҢзҡ„еӨ„зҗҶе’Ңе“Қеә”

locationиҜӯжі•

еҢ№й…ҚURLзұ»еһӢпјҢжңүеӣӣз§ҚеҸӮж•°еҸҜйҖүпјҢеҪ“з„¶д№ҹеҸҜд»ҘдёҚеёҰеҸӮж•°гҖӮlocation [ = | ~ | ~* | ^~ ] uri { вҖҰ }

  1. вҖң=вҖқ зІҫзЎ®еҢ№й…ҚпјҢеҶ…е®№иҰҒеҗҢиЎЁиҫҫејҸе®Ңе…ЁдёҖзӣҙжүҚеҢ№й…ҚжҲҗеҠҹ

  2. "~" жү§иЎҢжӯЈеҲҷеҢ№й…ҚпјҢеҢәеҲҶеӨ§е°ҸеҶҷ

  3. "~*" жү§иЎҢжӯЈеҲҷеҢ№й…ҚпјҢдёҚеҢәеҲҶеӨ§е°ҸеҶҷ

  4. "^~" иЎЁзӨәжҷ®йҖҡеӯ—з¬ҰдёІеҢ№й…ҚдёҠд»ҘеҗҺдёҚеҶҚиҝӣиЎҢжӯЈеҲҷеҢ№й…Қ

  5. дёҚеҠ д»»дҪ•и§„еҲҷж—¶пјҢй»ҳи®ӨжҳҜеӨ§е°ҸеҶҷж•Ҹж„ҹзҡ„пјҢеүҚзјҖеҢ№й…ҚпјҢзӣёеҪ“дәҺеҠ дәҶ"~"дёҺ"^~"

  6. "@" nginxеҶ…йғЁи·іиҪ¬

дҝ®йҘ°з¬Ұдјҳе…Ҳзә§

  1. =
  2. ж— 
  3. ^~
  4. ~жҲ–пҪһ*
  5. (ж— )

еҪ“з„¶иҝҷжҳҜзҗҶи®әдёҠзҡ„дёңиҘҝпјҢиҝҳжҳҜйңҖиҰҒе®һи·өжүҚеҸҜд»Ҙ
